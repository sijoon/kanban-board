# Kanban Board

To-do / In-progress / Done 3컬럼 칸반보드. 드래그앤드롭으로 카드를 이동할 수 있습니다.

## 기능

- 3컬럼 칸반보드 (To-do / In-progress / Done)
- 드래그앤드롭 카드 이동 (드롭 위치 미리보기)
- 카드 추가 / 삭제
- Supabase 이메일 / Google / GitHub 인증
- 모바일 반응형

## 기술 스택

- Vanilla HTML / CSS / JavaScript
- Supabase Auth
